# Crypto Adoption Forecasting with AI
- **Difficulty:** Intermediate (economics and data science)
- **Term Length:** Fall 2025

## Description
This project explores how macroeconomic stressors like inflation, unemployment, or interest rates impact cryptocurrency adoption. Students will combine macroeconomic indicators with blockchain usage data to forecast trends and visualize connections.

## Skills Gained
- Data collection from APIs like FRED and BLS
- Data wrangling and visualization in Python
- Forecasting models such as ARIMA and Prophet

## Technical Details
- **Data Analysis:** Python
- **Dashboard Tools:** Matplotlib, Seaborn, Streamlit, etc.
- **Data Integration:** Economic and blockchain datasets
